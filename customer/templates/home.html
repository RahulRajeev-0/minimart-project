API Endpoints Overview

Welcome to the MiniMart backend service. Below are the main endpoints available for integration with your frontend application.

Customer

Endpoints for Customer.

  • POST /customer/customers/

    Create Customer

  • PUT /customer/customers/{id}/

    Update Customer Details

  • DELETE /customer/customers/{id}

    Delete Customer

  • GET /customer/customers/{id}

    Get Customer Details

  • GET /customer/customers/

    List all Customer

Explore Customer API

Category

Endpoints for Category management.

  • GET /product/categories/

    List all categories

  • POST /product/categories/

    Create a new Category

  • PUT /product/categories/{id}/

    Update a category name

  • DELETE /product/categories/{id}/

    Delete a category

Explore Category API

Products

Endpoints for managing product inventory.

  • GET /product/products/

    List all products in inventory

  • GET /product/products/{id}/

    Get details for a specific product

  • POST /product/products/

    Add a new product

  • DELETE /product/products/{id}/

    Delete a product

  • PUT /product/products/{id}/

    Update a product

Explore Product API

Orders

Endpoints for order management and processing.

  • GET /order/orders/

    List all orders

  • GET /order/orders/{id}/

    Get details of an order

  • GET /order/customer-orders/{customer_id}/

    Get orders of a specific customer

  • POST /order/place-order/

    Create a new order or place order

  • DELETE order/order-delete/{id}/

    Delete order

  • PATCH /order/order-status/

    Update order status

Explore Orders API
